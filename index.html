Git starter commands

Starter commands

git init Initialise un repository local

git config --local user.name <user name> Initialise un nom d'utilisateur

git config --local user.email <user email> Initialise un email d'utilisateur

Add

git add <filename> Ajoute un fichier dans l'index

git add . Ajoute les fichiers du dossier en cours dans l'index

git add -A Ajoute tous les fichiers dans l'index (à ne pas utiliser de préférence)

Commit

git commit -m "your message" Commit des fichiers de la zone staging vers le repository

git commit -a Commit les fichiers de la zone staging vers le repository et ouvre l'éditeur par défaut pour éditer le message

Redact Commit message

  • Prendre le temps de rédiger son message
  • Ajouter un sujet
  • Limiter le message à 50 caractères
  • Ecrire sans accent
  • Répondre à la question : pourquoi ? plutôt que : comment ?

Affichage log

git log affiche les commits

git log -1 affiche le dernier commit

git log --pretty=oneline --max-count=2 affiche sur une ligne les 2 derniers commits

git log --pretty=oneline --since="5 minutes ago" affiche les derniers commits dans les 5 dernieres minutes

git log --oneline --until="5 minutes ago" affiche les derniers commits dans les 5 dernieres minutes

git log --oneline --author="Julian ALBINO" affiche les commits envoyés par Julian ALBINO (case sensitive)

git log --oneline --all --pretty=format:"%h %cd %s (%an)" --since="1 days ago" affiche tous les commits des dernières 24h

Format de Log

  • --pretty="..." : format de sortie
  • %h : abréviation hash code commit
  • %d : commit decoration
  • %ad : commit date
  • %cd : current date
  • %s : commentaires
  • %an : author name
  • --graph : commit tree
  • --date=...
